# Astrolabe built, then surface re-homed CDN → bundled React

**Date:** 2026-06-30

Built the **astrolabe** spell — a cross-project observatory conjuration
(singleton Bun daemon + thin cli + browser surface; layers = durable registry /
live presence = held SSE / current-only status) — via a lead-coordinated
multi-agent run. After a polish pass, an independent verify found the Tailwind
**Play CDN silently no-ops `@apply` component classes** (primary CTAs had
rendered as unstyled text since day one), so the surface was **re-homed
surface-only onto glamour's bundled React 19 + Bun + Tailwind v4 stack** as the
deliberate pilot for a unified spell-surface scaffold — the daemon and its 48
tests left untouched.

**Key files:**
`plugins/spellbook/skills/astrolabe/scripts/{server,cli,state}.ts` (daemon,
unchanged in re-home), `.../astrolabe/surface/**` (React surface),
`.../astrolabe/{bunfig.toml,tsconfig.json,SKILL.md}`
