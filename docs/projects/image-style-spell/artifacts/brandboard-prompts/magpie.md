Create a premium branding and visual identity board for a software application
called "Magpie".

Magpie allows users to upload a brand guide, mood board, style sheet,
presentation, or image and automatically extract all reusable assets.

Examples:

- logos
- icons
- illustrations
- color palettes
- typography references
- stickers
- UI elements

The extracted assets are exported as organized files with transparent
backgrounds.

BRAND IDEA

A magpie collects valuable things.

Magpie collects design assets.

The mascot should be a clever, friendly magpie bird that specializes in finding
useful visual elements.

STYLE

Modern design tool. Creative software. Professional but playful.

Think:

- Figma
- Canva
- Raycast
- Linear

but with a lovable mascot.
